Bluetooth Volume Manager может реагировать на подключение или отключение устройств Bluetooth.
Приложение позволяет системе Андроид запомнить громкость различных устройств Bluetooth.

Доступные возможности:
• Регулировка громкости музыки, вызовов и уведомлений.
• Отправка медиа-команд 'Воспроизведение' или 'Следующий'.
• Открытие конкретного приложения.
• Поддержка включённого состояния экрана.
• Предотвращение изменения громкости.
• Восстановление предыдущих уровней громкости после отключения.

Это приложение не может "увеличить" громкость или изменить громкость, если Вы не можете изменить её вручную с помощью кнопок громкости Вашего устройства.

Объяснение разрешений:
• 'Internet' для сообщений об ошибках.
• 'Bluetooth' для работы с устройствами Bluetooth.
• 'Modify audio settings' для изменения уровня громкости.
• 'Boot completed' для восстановления уровней громкости после перезагрузки.
• 'WAKE_LOCK' для исправления ошибки на устройствах Samsung и для предотвращения отключения экрана при необходимости.
• 'SYSTEM_ALERT_WINDOW' для запуска приложений при подключении Bluetooth-устройства (только на Android 10).
