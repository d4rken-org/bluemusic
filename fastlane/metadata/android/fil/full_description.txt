**Ang Bluetooth Volume Manager** ay naaalala ang mga antas ng volume para sa bawat isa sa iyong mga Bluetooth device at nagre-react kapag sila ay kumokonekta o nagdiskonekta.

**Mga Feature:**
* Itakda ang musika, tawag, ringtone, at notification volumes bawat device  
* Awtomatikong magpadala ng mga media command tulad ng Play o Next  
* Mag-launch ng mga app sa pagkonekta  
* Panatilihing gising ang screen  
* I-lock ang mga antas ng volume  
* I-restore ang volume pagkatapos ng disconnect  

Walang ads, walang tracking. Open-source.  
Ang ilang mga extras ay nangangailangan ng bayad na upgrade para suportahan ang development.

**Tandaan:** Hindi kayang palakasin ng app na ito ang volume o lampasan ang mga limitasyon ng system volume.

**Mga Pahintulot:**
* **Internet** – para sa mga opsyonal na bug report  
* **Bluetooth** – para ma-detect ang mga koneksyon ng device  
* **Mga setting ng audio** – para baguhin ang volume  
* **Boot** – para i-restore ang mga setting pagkatapos ng reboot  
* **Wake lock** – para panatilihing gising ang screen o ayusin ang mga bug ng Samsung  
* **System alert window** – para awtomatikong mag-launch ng mga app (Android 10 lamang)
