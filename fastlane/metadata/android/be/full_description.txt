**Bluetooth Volume Manager** запамінае узровні гучнасці для кожнай з вашых Bluetooth-прылад і рэагуе пры падключэнні або адключэнні.

**Функцыі:**
* Устанаўліваць гучнасць музыкі, званкоў, рынгтону і айных абвестак для кожнай прылады  
* Аўтаматычная адпраўка медыякаманд накшталт Пайграць або Наступны  
* Запуск праграм пры падключэнні  
* Трымаць экран уключаным  
* Блакаваць узровні гучнасці  
* Аднаўліваць гучнасць пасля адключэння  

Без рэкламы, без ссежання. Адкрыты код.  
Некаторыя дадатковыя функцыі патрабуюць платнага пашырэння для падтрымкі разработкі.

**Увага:** Гэта праграма не можа павялічыць гучнасць адбо абіходзіць сістэмныя абмежаванні гучнасці.

**Дазволы:**
* **Інтэрнэт** – для няабавязковых адправак аб памылках  
* **Bluetooth** – для вызначэння падключэнняў прылад  
* **Настройкі аўдыя** – для змянэння гучнасці  
* **Загрузка** – для аднаўлення настроек пасля перазагрузкі  
* **Wake lock** – для утрымання экрана ўключаным або выпраўлення памылак Samsung  
* **Сістэмнае акно** – для аўтазапуску праграм (толькі Android 10)
