**Bluetooth Volume Manager** запам'ятовує рівні гучності для кожного з ваших Bluetooth-пристроїв і реагує на їх підключення або відключення.

**Функції:**
* Встановлення гучності музики, дзвінків, рингтону та сповіщень для кожного пристрою  
* Автоматичне надсилання медіакоманд, таких як «Відтворити» або «Далі»  
* Запуск застосунків при підключенні  
* Утримувати екран увімкненим  
* Блокувати рівні гучності  
* Відновлення гучності після відключення  

Без реклами, без стеження. Відкритий код.  
Деякі додаткові функції потребують платного оновлення для підтримки розробки.

**Примітка:** Цей застосунок не може підвищити гучність або обійти системні обмеження гучності.

**Дозволи:**
* **Інтернет** – для необов'язкових звітів про помилки  
* **Bluetooth** – для виявлення підключень пристроїв  
* **Налаштування звуку** – для зміни гучності  
* **Завантаження** – для відновлення налаштувань після перезавантаження  
* **Блокування пробудження** – для утримання екрана увімкненим або виправлення помилок Samsung  
* **Системне вікно сповіщень** – для автоматичного запуску застосунків (лише Android 10)
