Цей додаток дозволяє Android запам'ятовувати гучність різних пристроїв Bluetooth.
Для кожного пристрою можливо налаштувати гучність як музики, так і виклику.
Часові параметри та конфігурація можуть бути підібрані індивідуально для кожного з пристроїв Bluetooth.

Тлумачення дозволів:

* 'Internet' для повідомлень про помилки.
* 'Bluetooth' для роботи з пристроями bluetooth.
* 'Modify audio settings' (Змінювати налаштування аудіо) для зміни гучності.
* 'Boot completed' (Завершене завантаження) для відновлення рівня гучності після перезавантажень.
* 'WAKE_LOCK' для усунення збою на пристроях Samsung, де цей дозвіл необхідний для запобігання помилці.
